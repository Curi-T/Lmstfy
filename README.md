### 简介
伸手党搜索教程专用，让我帮你搜索一下！  
项目非原创，仅仅为了个人的方便整合了在一起！  
更多详情访问：[来源项目](#来源项目)

新增 **支持跳转指定的网址**   
新增 **生成短链接**

如：输入 **如何百度url=curi-t.github.io**，  
将会跳转至 curi-t.github.io  
其中 **=** 符号可替换为 **@、#、&**  
即，指定跳转需要满足 **url(@|#|&|=)**
